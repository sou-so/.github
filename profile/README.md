## 프로젝트 소개

### 서비스 이름 : 🧡 소소 (Sou-so) 🧡

> 가사일에 지친 어머니들, 은퇴 후 삶이 무료한 아버지들을 겨냥한 동네 친구 만들기 플랫폼!

- **부제**: 마음만큼은 젊은 소년소녀들의 모임

- **타겟층**: 어머니, 아버지 세대의 중장년층 (40대~60대)

- **서비스 주요 기능**<br/>
  1️⃣ 8가지 카테고리를 주제로 커뮤니티 피드 생성<br/>
  2️⃣ 좋아요 & 북마크한 게시글 모아보기<br/>
  3️⃣ 게시글에 댓글, 대댓글 추가<br/>
  4️⃣ 채팅 (서비스 준비중)<br/>
  5️⃣ 소모임 (서비스 준비중)
